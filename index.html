Development Projects

Spotify Playlist Randomizer

A webapp that creates randomized Spotify playlists using Spotify listening history, favorite artists, and various other parameters. The app autheticates through OAuth, using Express and Node.js to handle authentication requests to the Spotify API and pull song recommendations. The frontend is built in React.js to display playlist generation and randomization, deployed on Heroku which the user can interact with.

Github Repo

Pleiades

Pleiades is a humorous dating application that matches users based on the positions of the stars. Data from a spreadsheet of astronomical data is fed it into Tangram’s deep learning environment to find matches and display them with a frontend built in Flask. Built in a duration of 72 hours for the PennApps XXII College Hackathon, Pleiades won multiple awards including "Best Machine Learning Hack."

Github Repo

Neural Network

A JavaFX program that uses deeplearning4j to train models and compare the differences in learning rate and accuracy that occur when the user adjusts things such as batch size, epochs, etc. The GUI allows the user to change these variables and then allows the user to visualize the differences between these different models by showing the difference in learning rate and accuracy for each run.

Github Repo

DDR FPGA

An implementation of the game ”Dance Dance Revolution” for the Basys 3 FPGA board, built in Verilog. The game parses input from an NES Controller and displays game progression visually using VGA output on a screen and lights on the Basys 3. The game also includes multiple increasing difficulties for a greater challenge.

Github Repo

Our Little Oasis

Our Little Oasis is a short 3d platformer game written in C# for the Unity engine. Being my first foray into solo game development, I faced numerous technical challenges centered around learning an entirely new engine and understanding the complete game development workflow from concept to completion. All assets (3d models, sound effects, music) were made by myself.
